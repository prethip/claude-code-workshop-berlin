# Portfolio template

The user is editing this template to make it their personal portfolio.

## Where things live

- **All content** is in `app/page.tsx` — headlines, paragraphs, projects, contact
- **Colors and fonts** are in `tailwind.config.ts`
- **Site title and meta description** are in `app/layout.tsx`

## How to make changes

- Edit the existing sections in `app/page.tsx` rather than adding new files.
- Use Tailwind utility classes — they're already configured.
- When adding a new section (e.g., "Experience"), follow the pattern of the existing sections: a `<section>` with a heading and content, separated by `mb-20`.
- Keep the visual style cohesive: clean, minimal, generous whitespace, single accent color.

## After each change

Remind the user to check the preview tab — Next.js auto-reloads, so they'll see the change immediately.
