# Portfolio template

A minimal personal portfolio. The main content lives in `app/page.tsx`.

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000 (or click the auto-forwarded port in Codespaces).

## What to ask Claude

Examples that work well:

- *"Change the headline to 'Hi, I'm Sara'"*
- *"Add a section called 'Experience' with 3 bullet points"*
- *"Make the accent color green instead of blue"*
- *"Add a link to my LinkedIn at the bottom"*
- *"Make the headline bigger and the spacing tighter"*

## Deploy to Vercel

```bash
vercel login
vercel --prod
```

Follow the prompts — accept the defaults. You'll get a URL like `your-portfolio.vercel.app`.
