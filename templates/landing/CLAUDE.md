# Landing page template

The user is editing this template to build a landing page for their small business.

## Where things live

- **All content** is in `app/page.tsx` — hero, services, contact
- **Colors and fonts** are in `tailwind.config.ts`
- **Site title and meta description** are in `app/layout.tsx`

## How to make changes

- Edit existing sections in `app/page.tsx` rather than adding new files.
- Use Tailwind utility classes — they're already configured.
- When adding a new section (e.g., "Testimonials", "Pricing"), follow the pattern of the existing sections: a `<section>` with a heading and content.
- Keep the visual style cohesive: confident hero, clear services, easy contact path.

## After each change

Remind the user to check the preview tab — Next.js auto-reloads, so they'll see the change immediately.
